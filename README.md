โครงการ “HandSense AI: ระบบแปลภาษามืออัจฉริยะ” เป็นโครงการที่มุ่งเน้นการพัฒนานวัตกรรม
ด้าน ปัญญาประดิษฐ์ (Artificial Intelligence) และ การประมวลผลภาพ (Computer Vision) เพื่อสร้าง
ระบบที่สามารถตรวจจับ จดจำ และแปลภาษามือให้เป็นข้อความภาษาไทยโดยอัตโนมัติ โดยใช้กล้องเป็น
อุปกรณ์อินพุตในการจับภาพท่าทางของมือและนิ้ว จากนั้นทำการประมวลผลด้วยโมเดล Deep Learning 
เพื่อจำแนกและถอดรหัสสัญญาณภาษามือให้เป็นข้อความที่เข้าใจได้ง่าย
